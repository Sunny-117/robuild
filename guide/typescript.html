TypeScript 支持 ​

robuild 提供完整的 TypeScript 支持，包括类型检查、声明文件生成和类型安全。

内置支持 ​

robuild 原生支持 TypeScript，无需额外配置：

bash
# 直接构建 TypeScript 文件
robuild ./src/index.ts

类型安全 ​

完整的类型定义 ​

robuild 提供完整的 TypeScript 类型定义：

typescript
import type {
  BuildConfig,
  BuildEntry,
  BundleEntry,
  TransformEntry,
  BuildContext,
  BuildHooks,
} from 'robuild'

// 类型安全的配置
const config: BuildConfig = {
  entries: [
    {
      type: 'bundle',
      input: './src/index.ts',
    } as BundleEntry,
  ],
}

高级特性 ​

类型推断 ​

robuild 会根据源码自动推断类型：

typescript
// 源码: src/utils.ts
export const VERSION = '1.0.0'
export const DEFAULT_CONFIG = {
  timeout: 5000,
  retries: 3,
} as const

export function createConfig(options: Partial<typeof DEFAULT_CONFIG>) {
  return { ...DEFAULT_CONFIG, ...options }
}
