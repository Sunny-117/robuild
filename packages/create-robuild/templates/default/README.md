# my-robuild-package

A TypeScript library built with [robuild](https://github.com/Sunny-117/robuild).

## Installation

```bash
npm install my-robuild-package
```

## Usage

```typescript
import { greet, add } from 'my-robuild-package'

console.log(greet('World')) // Hello, World!
console.log(add(1, 2)) // 3
```

## Development

```bash
# Install dependencies
npm install

# Build
npm run build

# Watch mode
npm run dev
```
