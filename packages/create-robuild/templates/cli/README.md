# my-cli

A CLI tool built with [robuild](https://github.com/Sunny-117/robuild).

## Installation

```bash
npm install -g my-cli
```

## Usage

```bash
my-cli greet World
# Hello, World!

my-cli greet World --uppercase
# HELLO, WORLD!
```

## Development

```bash
# Install dependencies
npm install

# Build
npm run build

# Run locally
npm start greet World
```
